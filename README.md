# Speedbridge Config
_Hey sh\*tass, wanna see me make a configuration API?_

**Speedbridge Config** is a configuration API that uses an annotation processor to generate a handler class at compile-time.  
No more configuration registries! No more reflection! No coding your own crummy framework! Just generate a handler and you're good to go!

For more information, see the [wiki](https://github.com/SpeedbridgeMC/config/wiki).

Javadocs are also available:
- [Annotations](https://javadoc.io/doc/io.github.speedbridgemc/config-annotations)
- [Runtime](https://javadoc.io/doc/io.github.speedbridgemc/config-runtime)
- [Processor](https://javadoc.io/doc/io.github.speedbridgemc/config-processor)
